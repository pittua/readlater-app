# 06. 開発ロードマップ

MVP までを小さく動かしながら積み上げる。各マイルストーンは「動くもの」を残す単位。

## M0. 環境・土台（プロジェクト雛形）
- [ ] Android Studio プロジェクト作成（Compose / Kotlin DSL / minSdk 26）
- [ ] バージョンカタログ（libs.versions.toml）で依存整理
- [ ] Hilt 導入、Application/Activity セットアップ
- [ ] Material3 テーマ・ナビゲーション骨組み
- [ ] Room DB（空のエンティティ＋AppDatabase）作成
- **完了条件**: 空のアプリが起動し、ナビゲーションで空画面が切り替わる。

## M1. 記事の保存（取り込みの心臓部）
- [ ] Article エンティティ / ArticleDao / Repository
- [ ] URL 正規化・重複判定
- [ ] 共有受信（ACTION_SEND インテントフィルタ）→ PENDING 保存
- [ ] OkHttp 取得 + Jsoup（OGP/メタ）+ Readability4J（本文）抽出
- [ ] FetchArticleWorker（WorkManager）で非同期取得 → SUCCESS/FAILED 更新
- **完了条件**: 他アプリの共有から URL を保存でき、本文・タイトル・サムネが入る。

## M2. 一覧・閲覧
- [ ] S-1 記事一覧（受信トレイ、Flow 購読、サムネ=Coil）
- [ ] 取得中/失敗の状態表示、再取得
- [ ] S-2 記事詳細（リーダービュー、抽出本文表示）
- [ ] 既読/アーカイブ/削除、スワイプ・長押しメニュー
- [ ] S-3 アプリ内 URL 追加ダイアログ
- **完了条件**: 保存→一覧→詳細で記事を読め、既読/アーカイブできる。

## M3. 整理（MVP の 2 本目の柱）
- [ ] Tag / Folder / ArticleTag エンティティ・DAO
- [ ] タグ付与・解除・サジェスト、S-5 タグ絞り込み
- [ ] フォルダ作成/リネーム/削除、S-4 フォルダ振り分け・閲覧
- [ ] S-6 検索（LIKE）
- **完了条件**: タグ・フォルダ・検索で目的の記事に到達できる。

## M4. 仕上げ
- [ ] S-7 設定（フォント/テーマ/自動既読/全削除）
- [ ] 空状態・エラー文言・オフライン挙動の調整
- [ ] 主要ロジックのテスト（抽出・DAO・ViewModel）
- [ ] アイコン・アプリ名確定・リリースビルド確認
- **完了条件**: 自分が日常利用できる完成度。

---

## マイルストーンと MVP スコープの対応

| スコープ（要件） | 対応 |
| --- | --- |
| URL 保存 / 記事取り込み | M1, M2(S-3) |
| 閲覧（リーダービュー） | M2 |
| タグ / フォルダ整理 | M3 |
| 検索 | M3 |
| 設定 | M4 |

## 次フェーズ候補（MVP 後）
- ハイライト・メモ（注釈）
- 全文検索（FTS）
- エクスポート/インポート → クラウド同期 → 複数端末
- RSS / ニュースレター取り込み、読み上げ(TTS)
